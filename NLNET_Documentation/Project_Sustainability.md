SHSM currently operates as an independent open-source project, maintained by an individual.
At this stage, the project's primary goal is to validate the design's feasibility and implement core functionalities, rather than building a large-scale community or commercial ecosystem.

# Maintenance Model
- The project is released under the AGPL V3 license, allowing anyone to freely use, modify, and distribute the code.
- The code is hosted on GitHub, and issues and pull requests are handled by the project maintainer (i.e., the developer).
- Currently, the project is in the Alpha phase, with maintenance focusing on feature enhancement and fixing known issues.

# Contribution Channels
- External developers can submit issue reports or pull requests via GitHub.
- As the project is still in its early stages, the contribution guidelines and developer documentation are not yet fully completed, but are planned for inclusion in the Beta phase.
- Contributions at this stage primarily rely on the maintainer's personal time, rather than an active network of external contributors.

# Future Sustainability Directions
If the project enters the Beta or official release phase, the following directions will be considered to enhance sustainability:

- Community Building: Gradually establish contributor guidelines to attract more developers to participate.
- Commercial Support Options: Provide commercial licensing options based on the AGPL V3 license to meet the needs of users requiring closed-source integration.
- Hosting Services: Consider providing a hosted version of SHSM as an optional service for user deployment.
- Partnerships: Explore the possibility of technical integration or cooperation with potential users or organizations.

# Current Phase Risk Assessment
| Risk | Current Status | Mitigation Measures |
|---|---|---|
| Limited Maintainer Time | The project is currently maintained by individuals in their spare time. | Limit the project scope, prioritize core functions; postpone the development of non-essential functions. |
| Lack of External Contributors | Currently, there are no active external contributors. | Provide contribution guidelines during the Beta phase; lower the participation threshold through documentation. |
| Uncertain Funding Support | NLnet grant application submitted, outcome pending. | Regardless of the funding outcome, the project will continue at the current pace.|

# Conclusion
Currently, the sustainability of SHSM depends on the continued dedication of individual developers. The project scope has been kept within what is achievable in the Alpha phase. With external support or funding, the project can accelerate its transition to the Beta phase and establish a more stable maintenance mechanism.
